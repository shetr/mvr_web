Spider AR

The goal of this project was to create mobile AR application, which takes input from camera, finds some specified mark and then displays 3D model on the position of the mark in the input image from camera. In this case 3D model of a spider was used. The spider is procedurally animated. As the spider's position changes, the spider moves with its legs accordingly. The application was made with Unity.
